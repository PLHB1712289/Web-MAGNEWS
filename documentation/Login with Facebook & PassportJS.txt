LOGIN WITH FACEBOOK & PASSPORTJS

B1: Tạo app FB trên trang "https://developers.facebook.com"
B2: Install thêm module "passport" và "passport-facebook"
B3: Tạo một module mới (new folder) để chứa các file config
B4: Config cho passport
B5: Thiết lập các router

Tài liệu tham khảo: https://viblo.asia/p/login-facebook-with-passport-on-nodejs-Do7546p0ZM6
